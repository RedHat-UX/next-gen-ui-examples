# Next Gen UI Examples

Repository contains example applications and extensions for [Next Gen UI Agent](https://github.com/RedHat-UX/next-gen-ui-agent)

## Examples

* [assistent-ui](./assistant-ui/) - Standalone chat UI powered by [Assistent UI](https://www.assistant-ui.com/)
* [streamlit](./streamlit/) - Standalone chat UI powered by [Streamlit](https://streamlit.io/)
* [rhds-hbc-example-renderer](./rhds-hbc-example-renderer/) - This custom renderer that extends RHDS one adds support for its own hand build components, giving a good example how users of Next Gen UI Agent can add support for their own HBCs to any existing renderer.
